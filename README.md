# Relational DB Design – Cleaning Products System (3NF)

## Overview
Relational database design and implementation for an automated cleaning products distribution system.

## What this project shows
- Requirements analysis
- Relational modeling and normalization approach
- Table creation with PK/FK relationships
- Data integrity through constraints

## Entities
- Proveedores (Suppliers)
- Productos (Products)
- Clientes (Customers)
- Ventas (Sales)
- Detalle_Venta (Sales detail)
- Entregas (Deliveries)

## Tools
SQL Server / T-SQL
